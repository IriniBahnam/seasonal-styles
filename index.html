Outfitter for All Seasons!

This webpage uses javaScript to demonstrate how to make the page more responsive. I implemented ancor tags for the user to click on which will navigate to a different look and feel of the page depending on the season they choose.

By clicking on each season the logo, image, header and background color will change to fit the desired outcome.

I used switch statements to execute a specific block of code depending on the case selected. I also included a defualt as shown to implement code if no seasons are clciked by the user.

This webpage is an example of some of the skills I have picked up along the way learning javaScript. I am now able to create a webpage that is dynamic and include all the features a customer requires.
